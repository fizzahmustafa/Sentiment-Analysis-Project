# AI Based Sentiment Analysis System

## Project Description
This project performs sentiment analysis on text using a LinearSVC machine learning model and TF-IDF vectorization.

## Features
- Positive, Negative, and Neutral sentiment prediction
- Flask API backend
- Streamlit dashboard
- Prediction history
- Text statistics

## Technologies Used
- Python
- Scikit-learn
- Flask
- Streamlit
- Pandas
- NumPy

## Model
- LinearSVC
- TF-IDF Vectorizer
